## Installation Notes

### macOS Security

If any of these programs give the error "unidentified developer" on macOS, see:
https://support.apple.com/guide/mac-help/open-a-mac-app-from-an-unidentified-developer-mh40616/mac

### Python Requirements

For Intention Repeater MAX GUI (Python version):

```bash
pip install -r requirements.txt
```

Note: Python is not needed if using the compiled executable version.
